# Datasets

This document contains a list of sources where you can find datasets for your project. You're also encouraged to bring your own!

> **Note for the data, make sure it doesn't include any personal, private, or sensitive information such as names, address, etc.**

- [UCI Machine Learning Repository](https://archive.ics.uci.edu/ml/datasets.php)
- [Azure Open Datasets](https://azure.microsoft.com/en-us/services/open-datasets/)
- [Microsoft Research Open Data](https://msropendata.com/)
- [Kaggle Datasets](https://www.kaggle.com/datasets)
- [Google Dataset Search](https://datasetsearch.research.google.com/)
- [United States Data Portal](https://www.data.gov/)
- [NYC Open Data](https://opendata.cityofnewyork.us/)
- [Chicago Data Portal](https://data.cityofchicago.org/)
- [San Francisco Open Data](https://datasf.org/opendata/)
- [Seattle Data Portal](https://data.seattle.gov/)
- [European Union Open Data Portal](https://data.europa.eu/euodp/en/data/)
- [Australian Data Portal](https://data.gov.au/)
- [New Zealand Data Portal](https://data.govt.nz/)
- [India Open Data Portal](https://data.gov.in/)
- [United States National Park Service Datasets](https://public-nps.opendata.arcgis.com/search)
- [World Bank Open Data](https://data.worldbank.org/)
- [Bank for International Settlements Datasets](https://www.bis.org/statistics/full_data_sets.htm)
- [United Nations Dataset](http://data.un.org/)
- [Harvard Dataverse](https://dataverse.harvard.edu/)
- [NASA Open Data](https://www.nasa.gov/open/data.html)
- [ESA Open Data](https://earth.esa.int/eogateway/)
